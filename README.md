# freshctl go-binary

Create a `.env_development.sh` file similar to the below.

```bash
export GCP_PROJECT_ID=aProjectId
export GCP_ZONE=aZone
export GCP_CLUSTER_NAME=aClusterName
```

Source the file.

```bash
source .env_development
```
